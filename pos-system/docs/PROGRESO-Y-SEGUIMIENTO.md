# PROGRESO Y SEGUIMIENTO

El progreso se calcula automáticamente con `scripts/progreso.js` y se adjunta como artefacto en CI (`progreso.yml`).

## Fuentes
- CI (lint/build/test/security)
- QA/E2E (Playwright)
- Contratos (comparación endpoints)
- Docker (build/push simulado)
- Tauri (installer RC/GA)
- Observabilidad (health/metrics)
- Docs (actualización)

## Artefactos
- `exports/status/progreso.json`
- `exports/status/progreso.md`
